How It Works (Guide)

Simple explanation for Admin and User. Demo-only pages.

For Admin

  1. Login to admin
  2. Enable modules (toggles)
  3. Connect APIs (demo configs)
  4. Activate Autopilot rules
  5. Monitor dashboard + logs

For User

  1. Call or place order
  2. Receive invoice/payment link
  3. Pay
  4. Track shipment
  5. Receive delivery

Safe Integration Example

Signals → publish event → Autopilot rule → action.
# 1) Publish an event (from any addon) publish_event_safe(event_key="storefront_order_paid", payload={"order_number":"SF-..."}, source="ecommerce_engine") # 2) Autopilot rule (admin-configured) event_key: storefront_order_paid actions: assign_courier, notify_customer, update_ledger
